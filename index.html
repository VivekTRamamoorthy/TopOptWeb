Structural Topology Optimisation

This is an interactive demo of structural topology optimisation on the web. This uses the solid isotropic material with penalisation (SIMP) method. The procedure here closely follows the 88-line matlab code by Andreassen et al.

Compliance: unknown
Iterations: 0
Time per iteration: unknown
Pick and paint:
void
solid
0.5
  • Use the buttons to run SIMP
  • You may manually change the shape by picking and painting void, solid, or 0.5 onto the elements, and run SIMP from that shape.
